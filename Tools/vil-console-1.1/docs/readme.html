Instructions:
  1. Unzip somewhere on your file system
  2. Start using Vil!
Tips:
  • Place Vil in your Path - so you can easily use it anywhere on the file system
  • You might also set up a shell/batch script or alias(if using Linux/Cygwin/etc.) to automate calls to Vil
  • Look at the Documentation and Metrics pages. The Documentation page includes some simple quick-start examples.
  • The download also includes the "Documentation" and "Metrics" pages.
  • Run vil against applications that are in a location where they have access to their dependent assemblies! Example: if you have an application that consists of 1 exe and 8 Dll's, and then copy the "exe" somewhere else and run vil against it, vil will not find the dependent dll's, and will be unable to perform its analysis.
  • Email if you have any problems or questions. Your email will get a response!
Constraints:
  • The download will perform analysis on assemblies totalling no more than 100 types(classes, enums, interfaces, and structs). If you need to perform analysis on assemblies totalling more then 100 types, just send an email. A key is available which can remove this constraint.
